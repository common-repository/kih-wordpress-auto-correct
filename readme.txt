Replace all occurrences of "wordpress" with the proper format

== Description ==

This simple plugin lets you correct all occurrences of "wordpress" in
your posts and pages with the proper "WordPress" spelling which is
with a capital "W" and capital "P"

== Installation ==

1. Unzip the plugin into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure in Wordpress under 'Plugins / Correct WordPress'
